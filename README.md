# The Assembled Thought

Showcase site for the paper **"Operator–Operand Factorization in LLM Residual Streams: Causal
Influence and Compositional Sufficiency"** (Matias Podeley, submitted to BlackboxNLP 2026).
Plain-language first, bilingual EN/ES, with two interactive canvas explorers driven by real
persisted model outputs — this repo needs **no GPU and no models**.

Live: **https://mpodeley.github.io/assembled-thought/**

The code, experiments, and reproducibility home is
[**jspace-qwen**](https://github.com/mpodeley/jspace-qwen) — every number on this site
regenerates from artifacts there. To learn the field in Spanish, see
[**Interpretabilidad Mecanicista**](https://mpodeley.github.io/interpretabilidad-mecanicista/).

## Build & deploy

```bash
pip install -r requirements.txt
mkdocs serve            # local preview (EN at /, ES at /es/)
mkdocs build --strict   # must pass clean
mkdocs gh-deploy --force
```
